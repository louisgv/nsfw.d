# nsfw.d

## Install

```bash
$ npm install nsfw.d
```


## CLI

```
$ nsfw.d --help
nsfw.d

Hello world command

Options:

	--name  Name of the person to greet           [string]
```


## Development

There are 2 available commands:

- `npm run dev` - Start development mode and recompile on change
- `npm run build` - Build a final distributable for npm
